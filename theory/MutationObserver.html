üìö Theory - MutationObserver Kya Hai?

MutationObserver ek Web API hai jo DOM mein hone wale changes ko detect karta hai.


Kab Use Karte Hain?

  • Jab koi element add/remove ho
  • Jab kisi element ka attribute change ho
  • Jab text content change ho
  • Third-party scripts ke changes track karne ke liye

üí° Syntax:

// Step 1: Observer banao
const observer = new MutationObserver(callback);

// Step 2: Observe karna shuru karo
observer.observe(targetElement, {
    childList: true,      // Child add/remove detect
    attributes: true,     // Attribute changes detect
    characterData: true,  // Text content changes detect
    subtree: true         // Nested elements bhi observe karo
});

// Step 3: Band karna ho toh
observer.disconnect();

üíª Complete Code Example

// 1. Target element select karo
const target = document.getElementById('targetElement');

// 2. Callback function banao - jab bhi change hoga yeh chalega
function handleMutations(mutationsList) {
    for (const mutation of mutationsList) {
        
        if (mutation.type === 'childList') {
            // Jab child add/remove ho
            mutation.addedNodes.forEach(node => {
                console.log('Added:', node);
            });
            mutation.removedNodes.forEach(node => {
                console.log('Removed:', node);
            });
        }
        
        if (mutation.type === 'attributes') {
            // Jab attribute change ho
            console.log('Attribute changed:', mutation.attributeName);
        }
        
        if (mutation.type === 'characterData') {
            // Jab text change ho
            console.log('Text changed:', mutation.target.textContent);
        }
    }
}

// 3. Observer create karo
const observer = new MutationObserver(handleMutations);

// 4. Observe karna shuru karo
observer.observe(target, {
    childList: true,
    attributes: true,
    characterData: true,
    subtree: true
});

// 5. Jab zaroorat na ho toh band karo
// observer.disconnect();
